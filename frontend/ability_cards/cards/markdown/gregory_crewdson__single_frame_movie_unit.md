# Gregory Crewdson｜SINGLE-FRAME MOVIE UNIT / 单帧电影剧组（案例级）

> 关键气质：**单张照片像一部电影的截图**。  
> 核心方法：输出剧照+幕后+通告单，强调“制作证据”。

## 0) 三件套
1. **电影式剧照成片（Tableau Still）**
2. **幕后片场照（Behind the Scene）**
3. **制作文件（Call Sheet / Lighting / Blocking）**

## 1) 归属牌组
- **展厅系｜单帧剧组包牌组（Single-Frame Movie Unit）**

## 2) Director_spec
- 题材：`郊区悬疑 / 夜间家庭 / 小镇静默`
- 光线：`雾化月光 / 钠灯 / 车灯切割`
- 规模：`小组 / 大组（更工业）`
- 风险：`不指涉真实事件；人物去身份化`

## 3) Visual_spec
- 成片：电影质感、低饱和、可疑的静止
- 幕后：灯架、吊臂、工作人员背影

## 4) Artifacts
1. **Call Sheet**（字段化）
2. **布光图说明**（文字）
3. **走位/道具清单**（可选）
4. **一句logline**（≤22字）

## 5) Workflow
1. 识别：光线戏剧化、空间阈值、叙事悬疑  
2. 配牌：单帧剧组包为主；辅以“灯箱叙事/静帧分镜页/冷笔记页”  
3. 导演：默认虚构；避免真实案件映射  
4. 生成：先出成片，再出幕后与文件  
5. 交付：附“虚构片场”声明
