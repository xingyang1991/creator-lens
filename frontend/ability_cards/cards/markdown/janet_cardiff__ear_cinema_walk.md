# Janet Cardiff｜EAR-CINEMA WALK / 耳机行走放映机（案例级）

> 关键气质：**你正在走路，但画面在你耳朵里放映**。  
> 核心方法：把单张图像延展为一个“听点路线”，用声音叠层制造幻觉。

## 0) 三件套
1. **路线谱（Walk Score）**：进入/偏离/回收 + 听点编号
2. **8个听点脚本（Cue Scripts）**：每点30-45秒
3. **双耳叠音表（Binaural Layers）**：L/R分别是什么

## 1) 归属牌组
- **放映系｜耳机行走谱牌组（Audio Walk）**

## 2) Director_spec
- 语气：`温柔引导 / 悬疑 / 自我对话`
- 速度：`慢走 / 正常 / 快走`
- 现实感：`更真实 / 更梦幻`
- 安全：`不包含追踪、定位、骚扰`

## 3) Visual_spec
- 路线谱：城市地图/建筑平面（虚构或模糊），路线用细线标注
- 现场照：人物背影戴耳机，环境与“对照照片卡”并置

## 4) Artifacts
1. **Walk Score**（图）
2. **Cue Scripts 8条**（听点编号+文本+环境声）
3. **Binaural Layers 表**（L/R）
4. **三张对照照片卡**（可选：你以为看到的 vs 实际看到的）

## 5) Workflow
1. 识别：空间强、动线感、门/走廊/街道、可形成“路线”  
2. 配牌：耳机行走谱为主；辅以“幻灯放映/静帧分镜页/冷笔记页”  
3. 导演：默认温柔引导；敏感地点要模糊化  
4. 生成：先出路线谱，再写8听点脚本  
5. 交付：附“虚构路线”声明
