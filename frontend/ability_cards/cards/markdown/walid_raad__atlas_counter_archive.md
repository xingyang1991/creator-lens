# Walid Raad｜ATLAS COUNTER-ARCHIVE / 幽灵档案卷（案例级）

> 关键气质：**档案既像真的，又总是漏一块**。  
> 核心方法：制造“FOUND/PRODUCED”反档案，含缺失文件通知与交叉索引。

## 0) 三件套
1. **反档案卷宗页（Archive Page）**：编号、元数据、遮蔽、交叉引用
2. **阅读室/展厅现场照（Context Shot）**
3. **缺失文件通知（Missing File Notice）**

## 1) 归属牌组
- **卷宗系｜幽灵档案卷牌组（Counter-Archive）**

## 2) Director_spec
- 文件身份：`FOUND / PRODUCED / DISPUTED`
- 口吻：`机构冷语 / 传闻 / 学术报告`
- 透明度：`明确虚构（默认） / 伪真（需风控）`
- 风险：`不冒充真实机构文件；不包含可用于定位的个人信息`

## 3) Visual_spec
- 档案页：印章、遮蔽条、编号、纸张磨损
- 缺失通知：像借阅单/失物招领

## 4) Artifacts
1. **文件摘要（Abstract）**
2. **分类索引卡（Index）**
3. **缺失文件通知（Notice）**
4. **讲座脚本（可选）**：作为“表演式档案”

## 5) Workflow
1. 识别：图像像“证物/档案/机构材料”，或可被伪造为档案  
2. 配牌：幽灵档案卷为主；辅以“伪档案柜/索引卡/系统脚注页”  
3. 导演：默认明确虚构  
4. 生成：先做档案页，再出缺失通知与索引  
5. 交付：附“反档案”说明与安全边界
