# 案例能力牌：ONE-YEAR PROTOCOL / 一年协议卡（Tehching Hsieh 向）

## 1) 归属牌组（固定掉落载体）
**牌组：连载账本牌组（Protocol + Grid + Gap Log）**  
固定掉落（不可漂移）：
- RULES合同页
- 24×N时间格可视化
- Gap Log 断点日志
- 档案墙存在场景

---

## 2) director_spec（0–2 个按钮 + 默认路径）
- 按钮1：协议：每小时打卡（默认）/ 每日一次誓言
- 按钮2：可见性：私密（默认）/ 公开展陈
- 默认：每小时 + 私密

---

## 3) visual_spec（主视觉 Image + 存在场景 Context Shot）

### 3.1 主视觉（Image）提示词
```text
RULES 5条（安全、制度口吻）+ 24×N网格（✔—✖示意）+ Gap Log 3条；输入照片仅作小证据碎片贴角；写免责声明：SIMULATION / NOT A HEALTH PROGRAM。
```

### 3.2 存在场景（Context Shot）提示词
```text
白墙挂满时间卡网格与少量小照片（去识别）；观众背影可选；档案氛围。
```

---

## 4) artifacts（主展品候选 2–4：名称 + 详细提示词）
### Artifact A（默认）｜Protocol Contract
```text
5条规则（安全可退出）+ 固定免责声明行。
```
### Artifact B｜Timecard Visualization
```text
grid_size/legend + 3条记录纪律（失败也入档）。
```
### Artifact C｜Gap Log
```text
3条断点：发生→记录→结构意义（中性）。
```

---

## 5) 一次完整 Session（Scan → Load → Direct → Reveal → Collect）
Scan：写进协议 → Load：协议/可见性 → Direct：2按钮 → Reveal：合同+网格+断点 → Collect：主展品=合同。

- 明确不鼓励现实模仿极端行为。
- 不提供健康/睡眠建议。
- 隐私遮蔽默认开启。
