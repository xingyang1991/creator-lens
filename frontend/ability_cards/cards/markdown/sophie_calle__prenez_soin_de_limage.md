# Sophie Calle｜PRENEZ SOIN DE L’IMAGE / 合唱卷宗（案例级）

> 关键气质：**把私人变成档案，但用合唱来分散责任**。  
> 核心方法：同一图像由多职业“证词合唱”解读，并严格区分事实与推测。

## 0) 三件套
1. **卷宗档案板主视觉（Dossier Board）**
2. **12位证词合唱（Chorus Statements）**
3. **事实/推测双栏（Fact vs Conjecture）**

## 1) 归属牌组
- **卷宗系｜合唱卷宗牌组（Dossier Chorus）**

## 2) Director_spec
- 职业池：`律师/花店/刑警/占星师/修复师/导游…`
- 口吻：`温柔 / 冷酷 / 幽默 / 迟疑`
- 透明度：`强区分事实/推测（默认） / 允许暧昧（需风控）`
- 风险：`不追踪、不猜测真实身份；可以虚构细节但必须标注`

## 3) Visual_spec
- 档案板：贴纸、遮蔽条、编号、回形针
- 展厅现场：照片与文本并列，像法庭证物但更文学

## 4) Artifacts
1. **Chorus 12条**（每条≤90字）
2. **Fact vs Conjecture 双栏表**（各8条）
3. **展签**（克制）
4. **封口便签**（可选：写“我不确定”）

## 5) Workflow
1. 识别：人物/关系/私密空间/可被多视角解释  
2. 配牌：合唱卷宗为主；辅以“冷笔记页/伦理脚注页/证据墙”  
3. 导演：默认强区分事实/推测  
4. 生成：先出合唱，再做档案板视觉  
5. 交付：附去识别与免责声明
