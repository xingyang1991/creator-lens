# Roman Opalka｜INFINITE COUNT / 无尽计数进度条（案例级）
生成日期：2025-12-28

## 研究要点（方法与媒介纪律）
- 连续计数绘画：每张Detail延续上一张；位置纪律（左上→右下、行列）。
- 序列与编号把生命变成可见进度条。
- 本卡输出数字表皮+起止号+用时+唯一旁注，天然连载。

# 案例能力牌：INFINITE COUNT / 无尽计数进度条（Roman Opalka 向）

## 1) 归属牌组（固定掉落载体）
**牌组：连载账本牌组（Number Surface + Start/End + Note）**  
固定掉落（不可漂移）：
- 数字表皮主视觉
- START/END + DURATION
- 唯一旁注
- 肌理近景存在场景

---

## 2) director_spec（0–2 个按钮 + 默认路径）
- 按钮1：速度：慢（默认）/ 快
- 按钮2：底色：不推进（默认）/ 向白渐进
- 默认：慢 + 不推进

---

## 3) visual_spec（主视觉 Image + 存在场景 Context Shot）

### 3.1 主视觉（Image）提示词
```text
连续数字覆盖画面（左上起、行列推进）；右侧START/END/DURATION（记录口吻）；输入照片仅作底纹；底部一句旁注≤18字；隐私遮蔽。
```

### 3.2 存在场景（Context Shot）提示词
```text
近拍数字肌理+极简展签（Detail No./Start-End/Year）。
```

## 5) 一次完整 Session（Scan → Load → Direct → Reveal → Collect）
Scan：继续计数 → Load：速度/底色 → Direct：2按钮 → Reveal：表皮+字段+规则 → Collect：主展品=计数账。
