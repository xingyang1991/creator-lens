# Amalia Ulman｜SCRIPTED FEED / 演出时间线生成器（案例级）

> 关键气质：**社交媒体是一场可编排的演出**。  
> 核心方法：生成一个“虚构表演的时间线”，并在最后揭示其为作品。

## 0) 三件套
1. **Mock Feed 主视觉**：时间线截屏样式（必须带“FICTIONAL PERFORMANCE”水印）
2. **三幕脚本（7-12帖）**：每帖含图片意图+文案+发布时间
3. **文案账本（Caption Ledger）**：词汇池、语气规则、emoji策略

## 1) 归属牌组
- **皮肤系｜演出时间线牌组（Scripted Feed）**

## 2) Director_spec
- 叙事：`上升/崩塌/回归`（三幕）
- 语气：`甜 / 冷 / 自嘲 / 过度真诚`
- 真实性：`更像真的 / 更像作品`
- 风险：`不得模仿真实个人账号、不得教唆欺诈；保持“表演件”属性`

## 3) Visual_spec
- 时间线UI：头像、点赞、评论、时间戳、地点（地点默认模糊或虚构）
- 水印：FICTIONAL PERFORMANCE / ART WORK

## 4) Artifacts
1. **时间线脚本（9帖标准）**
2. **文案账本**（规则+词汇池）
3. **评论合唱**（可选：8条，不做真实用户模拟）
4. **揭示卡**（最后一帖：这是作品的声明）

## 5) Workflow
1. 识别：自拍/生活照/品牌感/社交气质/可做时间线  
2. 配牌：演出时间线为主；辅以“平台皮肤页/讽刺头版/合唱卷宗”  
3. 导演：默认“更像作品”；若“更像真的”需加大水印与免责声明  
4. 生成：先出脚本，再出视觉  
5. 交付：严格的风控与去身份化
