# Susan Sontag｜ETHICS OF SEEING / 观看伦理脚注（案例级）
生成日期：2025-12-28

## 研究要点（工作方式 / 特征 / 输出形式 / 风格）
- 《On Photography》提出摄影的权力性：例如“there is an aggression implicit in every use of the camera”，把拍摄视为一种占有/消费世界的方式。
- 《Regarding the Pain of Others》强调观看他者痛苦的伦理复杂性：图像既可能唤起注意，也可能导致麻木；关键在于我们如何解释与使用这些图像。
- 本能力转译：把照片编译成“观看伦理注释页”：观察→伦理问题→两条规则→不确定性；输出是作品化批评，不是道德审判。

# 案例能力牌：ETHICS OF SEEING / 观看伦理脚注（Susan Sontag 向）

## 1) 归属牌组（固定掉落载体）
**牌组：文档系｜伦理脚注牌组（Observation + Questions + Rules + Reflection）**  
固定掉落（不可漂移）：
- 主视觉：伦理注释页（照片小贴附+编号）
- 右侧：3个问题（观看/权力/责任）
- 底部：why_rules ≤2（观看规则）+ Reflection（不确定性与自我质疑）
- 存在场景：阅读室/研讨会讲义

---

## 2) director_spec（0–2 个按钮 + 默认路径）
- 按钮1｜**观看对象**：日常消费（默认）/ 他者痛苦 / 权力景观
- 按钮2｜**语气**：怀疑（默认）/ 更锋利
- 默认路径：日常消费 + 怀疑

---

## 3) visual_spec（主视觉 Image + 存在场景 Context Shot）

### 3.1 主视觉（Image）提示词
```text
把输入照片编译成“观看伦理讲义页”：
- 照片以小图贴附（裁切避开人脸/门牌），并标注1–3个编号点指向细节（反光、遮挡、边缘、视线方向）
- 右侧写3个问题：
  Q1 观看在这里如何占有？
  Q2 谁被变成对象？谁获得权力？
  Q3 这张图将被如何消费/传播？
- 底部写why_rules ≤2：两条观看规则（例如：先承认位置；再决定是否传播）
- 加Reflection 1段（60–90字）：承认不确定性与自我质疑（不下道德判决）
- 去识别与去定位默认开启
```

### 3.2 存在场景（Context Shot）提示词
```text
生成“研讨会讲义存在场景”：
- 桌面上摊开一叠讲义/书页，旁有笔记与荧光笔
- 可见本次伦理注释页缩略
- 氛围：阅读、讨论、克制
```

## 4) artifacts（主展品候选 2–4：名称 + 详细提示词）
### Artifact A（默认主展品）｜Ethical Questions 3
```text
写3个问题：必须具体指向照片的某个细节（边缘/遮挡/反光/视线），并逼问观看关系。
```

### Artifact B｜Rules ≤2（观看规则）
```text
两条规则：
- 规则1：承认观看位置（我是谁/我在做什么）
- 规则2：决定传播边界（什么必须遮蔽/什么不该发布）
语气克制，不是训斥。
```

### Artifact C｜Reflection Note（反思旁注）
```text
60–90字：承认图像既能唤起注意也能导致麻木；写出你愿意承担什么、不愿意承担什么。
```

---

## 5) 一次完整 Session（Scan → Load → Direct → Reveal → Collect）
1) Scan：判词=“摄影是一种关系”；证据=3个观看线索
2) Load：观看对象/语气推荐
3) Direct：0–2按钮
4) Reveal：先出3问句，再补规则与反思
5) Collect：默认主展品=Ethical Questions 3

---

## 6) 安全与边界（必须遵守）
- 不把特定受害者/弱势群体做猎奇呈现。
- 不输出可识别隐私；去识别去定位默认开启。
- 输出是作品化批评与反思，不作现实指控。
