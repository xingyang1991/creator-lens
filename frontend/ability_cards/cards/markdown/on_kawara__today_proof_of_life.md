# On Kawara｜TODAY/PROOF OF LIFE / 日期证明板（案例级）

> 关键气质：**今天存在的证明**。  
> 核心方法：把图像转为“日期板 + 盒内剪页 + 当日单据”的档案组合。

## 0) 三件套
1. **日期板主视觉（Date Painting）**：当日日期（当地语言/格式）
2. **档案盒剪页（Archive Box Liner）**：当日报纸/网页标题剪页（可虚构/抽象）
3. **当日派生单据（Daily Slip）**：I GOT UP / I WENT / ALIVE（任选其一）

## 1) 归属牌组
- **连载系｜日期证明板牌组（Date Board）**

## 2) Director_spec
- 格式：`YYYY.MM.DD / 29 DEC. 2025 / 本地语言`
- 证据类型：`报纸剪页 / 浏览器标题 / 收据碎片`
- 情绪：`无情绪 / 轻微讽刺 / 极克制`
- 隐私：`不出现真实地址与可追踪信息`

## 3) Visual_spec
- 日期板：深色纯底，白字日期，边缘有油画肌理
- 盒内：纸张贴附、编号、夹子
- 单据：极简、像收据或电报

## 4) Artifacts
1. **Why/Rules**（纪律声明≤2）
2. **当日剪页条目**（3条：标题/来源类型）
3. **Daily Slip**（字段化：时间/动作/地点模糊）
4. **序列编号**（如 #2025-12-29）

## 5) Workflow
1. 识别：图像具有“日常纪念/证明/清单”潜力  
2. 配牌：日期证明板为主；辅以“年度协议/无尽计数/索引卡”  
3. 导演：默认无情绪；不做精确定位  
4. 生成：先做日期板，再做剪页与单据  
5. 交付：形成可长期连载模板
