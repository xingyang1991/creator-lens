# Maurizio Cattelan｜BREAKING LOT / 机构新闻包（案例级）

> 关键气质：**作品化的丑闻结构**（但必须是虚构、无伤害、无违法）。  
> 核心方法：把图像包装成“机构事件”的媒体资料包。

## 0) 三件套
1. **新闻照式主视觉（Press Photo）**
2. **拍卖图录条目（Lot Sheet）**
3. **证书+安装示意（COA + Diagram）**

## 1) 归属牌组
- **展厅系｜机构新闻包牌组（Institutional Press Kit）**

## 2) Director_spec
- 机构类型：`美术馆 / 拍卖行 / 品牌基金会`
- 口吻：`官方克制 / 暗自兴奋 / 危机公关`
- 风险：`不得教唆违法；不得针对真实个人；必须虚构机构名`

## 3) Visual_spec
- 新闻照：闪光、记者角度、围观
- 预展现场：白盒子、警戒线、工作人员背影

## 4) Artifacts
1. **Lot Sheet**（字段：标题、年份、材料、尺寸、估价、备注）
2. **COA**（证书字段化）
3. **安装示意**（文字版）
4. **媒体快讯标题**（可选：3条）

## 5) Workflow
1. 识别：图像具有“事件性/可被机构包装”的潜力  
2. 配牌：机构新闻包为主；辅以“灯箱叙事/标题机关/讽刺头版”  
3. 导演：默认危机公关口吻；高风险需降锋利度  
4. 生成：先做新闻照，再做图录与证书  
5. 交付：明确虚构与安全边界
