# René Magritte｜THIS IS NOT / 标题机关绘画（案例级）
生成日期：2025-12-28

## 研究要点（工作方式 / 特征 / 输出形式 / 风格）
- 方法特征：用清晰、广告般的绘画语言呈现普通物，但通过语境错位与标题/文字机关制造哲学冲突（词/像/物不可重合）。
- 输出形式稳定：干净、平涂、边界清晰；标题与图像互相打架，作品像一个“逻辑陷阱”。
- 本能力转译重点：把照片编译成“概念画+一句机关标题+两条悖论规则”，让作品感来自语境先行而非滤镜。

# 案例能力牌：THIS IS NOT / 标题机关绘画（Magritte 向）

## 1) 归属牌组（固定掉落载体）
**牌组：展陈系｜概念绘画牌组（Clean Painting + Caption Trap + Rules）**  
固定掉落（不可漂移）：
- 主视觉：平涂干净的“概念画”
- 右侧：Caption Trap（标题机关3个备选）
- 底部：Paradox Rules ≤2 + 展签
- 存在场景：画框上墙+展签

---

## 2) director_spec（0–2 个按钮 + 默认路径）
- 按钮1｜悖论类型：词像冲突（默认）/ 尺度错位 / 物体替换
- 按钮2｜露馅程度：微妙（默认）/ 明显
- 默认：词像冲突 + 微妙

---

## 3) visual_spec（主视觉 Image + 存在场景 Context Shot）

### 3.1 主视觉（Image）提示词
```text
把输入照片编译成“干净平涂的概念画”（不是照片滤镜）：
- 物体/场景保持写实清晰，但去掉摄影质感，改为平涂、边界干净、光线均匀
- 添加一行标题机关（中文为主）：结构类似“这不是X”/“这不是一张照片”/“这不是…而是…”（不要复刻法语原句作为固定文本）
- 悖论必须成立：标题与画面在逻辑上冲突或互相否定
- 右侧给3个标题备选（短、像广告口号）
- 底部写Paradox Rules ≤2：规则声明而非解释
- 去识别/去定位默认开启
```

### 3.2 存在场景（Context Shot）提示词
```text
生成存在场景：
- 白墙展厅中框装一幅画，旁边小展签
- 展签只写作品名/年份/媒介（oil on canvas style）
- 观众背影可选（不可识别）
```

---

## 4) artifacts（主展品候选 2–4：名称 + 详细提示词）
### Artifact A（默认主展品）｜Caption Trap 3（标题机关备选）
```text
3个标题：每个≤12字，必须让词/像/物产生冲突；至少1个是否定句。
```

### Artifact B｜Paradox Rules ≤2（悖论规则）
```text
两条规则：
- 规则1：你看到的不是你以为的那个类别
- 规则2：文字不是说明，而是陷阱
语气像声明，不解释道理。
```

### Artifact C｜Wall Label（展签）
```text
120–150字：只讲“词像物的不可重合”与“广告式清晰为何更危险”；最后一句把观众放进陷阱。
```

---

## 5) 一次完整 Session（Scan → Load → Direct → Reveal → Collect）
1) Scan：判词=“语境反转”；证据=3个可被否定的对象
2) Load：悖论类型/露馅程度推荐
3) Direct：0–2按钮
4) Reveal：先出主视觉概念画，再补标题备选与规则
5) Collect：默认主展品=Caption Trap 3

---

## 6) 安全与边界（必须遵守）
- 不复刻具体名作图像与固定台词；只致敬方法。
- 去识别/去定位默认开启。
- 避免输出仇恨与冒犯内容。
