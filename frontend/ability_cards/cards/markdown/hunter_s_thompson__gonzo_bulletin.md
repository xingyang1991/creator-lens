# Hunter S. Thompson｜GONZO BULLETIN / 贡佐快讯（案例级）
生成日期：2025-12-28

## 研究要点（方法与媒介纪律）
- 贡佐新闻：记者置于故事中心，主观参与式，讽刺与夸张并用。
- 与插画语言结合形成强视觉（本卡用抽象墨迹/手绘痕迹皮肤）。
- 本卡必须原创、去定位、禁止危险行为鼓励。

# 案例能力牌：GONZO BULLETIN / 贡佐快讯（HST 向）

## 1) 归属牌组（固定掉落载体）
**牌组：媒体皮肤牌组（First-Person Bulletin + Footnotes）**  
固定掉落（不可漂移）：
- 专栏页主视觉（标题+三段正文）
- 比喻清单5条
- 脚注3条（自嘲/不可靠）
- 杂志页存在场景

---

## 2) director_spec（0–2 个按钮 + 默认路径）
- 按钮1：对象：权力机器（默认）/ 消费狂欢 / 自我崩坏
- 按钮2：失控：轻度亢奋（默认）/ 高压混沌
- 默认：权力机器 + 轻度亢奋

---

## 3) visual_spec（主视觉 Image + 存在场景 Context Shot）

### 3.1 主视觉（Image）提示词
```text
专栏页：标题≤14字+副标题；正文3段（现场/制度/自嘲）；右侧比喻5条；脚注3条承认不可靠；边缘抽象墨迹手绘痕迹；声明原创模拟与安全；隐私遮蔽。
```

### 3.2 存在场景（Context Shot）提示词
```text
桌面摊开杂志/报纸专栏页，旁有咖啡渍与涂写；可见标题与插画痕迹。
```

---

## 4) artifacts（主展品候选 2–4：名称 + 详细提示词）
### Artifact A｜Bulletin Copy
```text
标题+副标题+三段正文（每段≤70字）；第一人称；不点名真实个人与地点；原创。
```
### Artifact B｜Metaphor List 5
```text
5条制度怪物比喻；禁止仇恨与针对群体攻击。
```
### Artifact C｜Footnotes 3
```text
3条脚注：自嘲/怀疑自己/承认夸张。
```

---

## 5) 一次完整 Session（Scan → Load → Direct → Reveal → Collect）
Scan：现场 → Load：对象/失控 → Direct：2按钮 → Reveal：标题→正文+比喻+脚注 → Collect：主展品=标题+副标题。

---

## 6) 安全与边界（必须遵守）
- 不引用原作句子。
- 不鼓励药物/暴力/危险驾驶等；不提供操作细节。
- 去定位与隐私遮蔽默认开启。
