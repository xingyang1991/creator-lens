# Martin Parr｜SUNBURNT SOCIOLOGY / 饱和闪光社会学（案例级）
生成日期：2025-12-28

## 研究要点（工作方式 / 特征 / 输出形式 / 风格）
- 典型工作方式：以“日常人类学”观察阶层、消费与休闲；常用饱和色彩与直打闪光（on-camera flash）让肤色、塑料、油渍与尴尬细节成为叙事。
- 输出形式强系列化：海滨/旅游/购物/日常过剩被做成可连载的‘社会标本’，标题与编辑顺序本身构成观点。
- 本能力转译重点：照片先被编译为“海滩/消费社会标本卡”，再输出“细节清单+阶层信号地图+一句冷笑展签”。

# 案例能力牌：SUNBURNT SOCIOLOGY / 饱和闪光社会学（Parr 向）

## 1) 归属牌组（固定掉落载体）
**牌组：摄影牌组（Saturated Flash Still + Anthropological Caption + Detail Inventory）**  
固定掉落（不可漂移）：
- 主视觉：饱和彩色+直打闪光的“日常标本照”
- 右侧：Detail Inventory（12条细节清单）
- 底部：Class Signals Map（3条‘阶层/消费信号’）+ Wall Label（冷静又微嘲）
- 存在场景：摄影书双页/展厅系列挂法

---

## 2) director_spec（0–2 个按钮 + 默认路径）
- 按钮1｜场域：海边/休闲（默认） / 购物/消费
- 按钮2｜温度：同情式冷眼（默认） / 更尖刻（危险槽：自动弱化对个体的指向，转向制度/消费机制）
- 默认：海边/休闲 + 同情式冷眼

---

## 3) visual_spec（主视觉 Image + 存在场景 Context Shot）

### 3.1 主视觉（Image）提示词
```text
把输入照片编译成“饱和闪光的日常标本照”：
- 色彩更饱和但不梦幻；闪光要硬，质地要真实（皮肤油光/塑料反光/食物油渍/衣料纹理）
- 构图要“近距离但不亲密”：让尴尬细节露出来
- 画面必须留出‘可读的小道具/小动作’（手势、包装、配饰、垃圾、标识残片）
- 默认去识别去定位：人脸可模糊或裁切；门牌车牌地标文字遮蔽
- 在画面角落加极小的系列编号：VOL.#（用于连载）
- 禁止美颜/滤镜化叙事
```

### 3.2 存在场景（Context Shot）提示词
```text
生成“摄影书双页/展厅系列”存在场景：
- 摄影书摊开：左页大图为本次主视觉，右页是‘细节清单+阶层信号’
- 或白墙上同系列3张并列挂法（本次主视觉在中）
- 观众背影可选（不可识别面部）
```

---

## 4) artifacts（主展品候选 2–4：名称 + 详细提示词）
### Artifact A（默认主展品）｜Detail Inventory 12（12条细节清单）
```text
12条：每条=细节名词（≤6字）+ 证据（指向照片具体位置/反光/边缘/遮挡）+ “它透露什么”（消费/阶层/体面/疲惫之一）。
禁止对个体道德审判。
```

### Artifact B｜Class Signals Map（阶层/消费信号 3 条）
```text
3条：信号→它被谁使用→它如何被误读（把矛盾写出来）。
```

### Artifact C｜Wall Label（展签）
```text
120–160字：把它写成“社会标本”而不是风景照；最后一句把观众放入凝视机制（你也在看，也在被看）。
```

---

## 5) 一次完整 Session（Scan → Load → Direct → Reveal → Collect）
1) Scan：判词=“这是社会标本”；证据=3个‘小道具’
2) Load：场域/温度推荐
3) Direct：0–2按钮
4) Reveal：先出主视觉，再补细节清单与信号地图
5) Collect：默认主展品=Detail Inventory 12

---

## 6) 安全与边界（必须遵守）
- 不对现实个体做羞辱式描写；尖刻模式自动转向制度/消费结构。
- 去识别/去定位默认开启。
- 不复刻具体作品与标题，只致敬方法与语法。
