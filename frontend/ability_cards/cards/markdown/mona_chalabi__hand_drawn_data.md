# Mona Chalabi｜HAND-DRAWN DATA / 手绘数据判词（案例级）
生成日期：2025-12-28

## 研究要点（方法与媒介纪律）
- 手绘让“人的决策”可见：选择、取舍、立场与类比一起成为视觉语言。
- 强调可理解性与不确定性：用对照与尺度把数字放进人类经验。
- 本卡掉落“可截图传播的手绘数据页”，必须标注示意/假设，避免伪造事实。

# 案例能力牌：HAND-DRAWN DATA / 手绘数据判词（Mona Chalabi 向）

## 1) 归属牌组（固定掉落载体）
**牌组：文档系｜手绘数据页牌组（Sketch Chart + Notes + Uncertainty）**  
固定掉落（不可漂移）：
- 手绘图表主视觉（纸纹+手绘线条）
- 注释3条（类比/尺度/立场）
- Uncertainty Note（假设与不确定性）
- 社媒卡/墙贴存在场景

---

## 2) director_spec（0–2 个按钮 + 默认路径）
- 按钮1：图表：对比条形（默认）/ 比例饼图 / 分布点图
- 按钮2：语气：温柔解释（默认）/ 冷刺一针
- 默认：对比条形 + 温柔解释

---

## 3) visual_spec（主视觉 Image + 存在场景 Context Shot）

### 3.1 主视觉（Image）提示词
```text
纸纹背景+手绘图表；标题≤18字必须“把数字放进人类尺度”；右侧3条手写注释；底部Uncertainty Note 1–2句；角落标“示意/模拟数据”；不出现真实账号与可定位地址。
```

### 3.2 存在场景（Context Shot）提示词
```text
笔记本夹页/墙贴/社媒卡片式展示（非真实平台UI）；可见胶带与便签；不出现真实用户名。
```

---

## 4) artifacts（主展品候选 2–4：名称 + 详细提示词）
### Artifact A（默认）｜Chart + One-liner
```text
标题≤18字；5–7类示意数值（标注示意）；结论一针≤14字。
```
### Artifact B｜Annotation Notes 3
```text
3条：直觉→对照→为什么重要（社会/权力/资源）。
```
### Artifact C｜Uncertainty Note
```text
写清假设与可能改变结论的变量（克制、诚实）。
```

---

## 5) 一次完整 Session（Scan → Load → Direct → Reveal → Collect）
Scan：把数字变成人类尺度 → Load：图表/语气 → Direct：2按钮 → Reveal：图表+注释+不确定性 → Collect：主展品=图表。

---

## 6) 安全与边界（必须遵守）
- 必须标注示意/假设，不伪装真实统计。
- 禁止仇恨/骚扰/煽动。
- 去定位与隐私遮蔽默认开启。
