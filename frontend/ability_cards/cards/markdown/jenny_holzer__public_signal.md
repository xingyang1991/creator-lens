# Jenny Holzer｜PUBLIC SIGNAL / 公共标语投放机（案例级）

> 关键气质：**短句、公共介质、冷硬口吻**。  
> 核心方法：把图像中的情绪或矛盾写成可投放的“标语阵列”。

## 0) 三件套
1. **公共介质主视觉**：LED/投影/海报/路牌（任选其一）
2. **标语阵列（Truisms）**：12或24条原创短句
3. **播放/张贴脚本（Cue Sheet）**：速度、停顿、循环、节奏

## 1) 归属牌组
- **展厅系｜公共标语机牌组（Public Signal / Truisms）**

## 2) Director_spec
- 语气：`冷硬命令 / 低声告白 / 法律条款口吻`
- 介质：`LED / 投影 / 海报 / 路牌`
- 强度：`温和 / 辛辣 / 危险（需风控）`
- 禁区：`不针对具体个人、群体仇恨、暴力号召`

## 3) Visual_spec
- LED：黑底发光字、公共空间夜景
- 投影：墙面或建筑外立面，字被环境打断
- 海报：网格排版，留白，像公共公告

## 4) Artifacts
1. **Truisms 阵列**（12/24）
2. **Cue Sheet**（每条：停顿时长、亮度、滚动速度）
3. **场地说明**（3种投放场景）
4. **伦理声明**（为什么不点名）

## 5) Workflow
1. 识别：画面里有冲突/态度/矛盾，或可提炼成一句话  
2. 配牌：公共标语机为主；辅以“标题机关/讽刺头版/伦理脚注页”  
3. 导演：强度默认“辛辣以下”，危险档需要人工确认  
4. 生成：先出短句阵列，再做介质可视化  
5. 交付：附禁用清单与免责声明
