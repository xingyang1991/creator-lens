# 案例能力牌：MNEMOSYNE PANEL / 记忆图谱板（Warburg 向）

## 1) 归属牌组（固定掉落载体）
**牌组：展陈系｜黑布面板牌组（Black Panel + Clusters + References）**  
固定掉落（不可漂移）：
- 黑布面板主视觉（Panel编号）+ 12/24片段
- Cluster Map 3组
- Cross-References 3条 + WIP声明
- 研究室存在场景

---

## 2) director_spec（0–2 个按钮 + 默认路径）
- 按钮1：主题：姿态（默认）/ 权力符号 / 商品广告
- 按钮2：密度：12（默认）/ 24
- 默认：姿态 + 12

---

## 3) visual_spec（主视觉 Image + 存在场景 Context Shot）

### 3.1 主视觉（Image）提示词
```text
黑布面板Panel No.XX；钉12或24片段（≥6来自输入图裁切）；其余用抽象占位或用户自有素材；细线形成3–5关联链；右侧主题词与姿态词；保留空位/贴纸显示WIP；隐私遮蔽。
```

### 3.2 存在场景（Context Shot）提示词
```text
研究室墙面挂黑布面板，旁有文件夹与更多面板；不必出现人物。
```

---

## 4) artifacts（主展品候选 2–4：名称 + 详细提示词）
### Artifact A（默认）｜Cluster Map
```text
3组簇：title/items/linking_gesture/reorder_effect。
```
### Artifact B｜Cross-References
```text
3条：REF PANEL / shared motif / missing link。
```
### Artifact C｜WIP Note
```text
60–90字：声明持续重排、不下结论；把观看者变成共同编辑者。
```

---

## 5) 一次完整 Session（Scan → Load → Direct → Reveal → Collect）
Scan：图谱节点 → Load：主题/密度 → Direct：2按钮 → Reveal：面板+簇/引用/WIP → Collect：主展品=簇地图。

---

## 6) 安全与边界（必须遵守）
- 外部图像仅占位符或用户自有素材。
- 不把联想当事实。
- 隐私遮蔽与去定位默认开启。
