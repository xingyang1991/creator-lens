# Joan Didion｜CALM KNIFE / 冷刀笔记（案例级）
生成日期：2025-12-28

## 研究要点（方法与媒介纪律）
- 从笔记本抓取细节与对话碎片，以冷静句法切割观察。
- 笔记是私人意义索引而非事实档案，这使“作品卡=私人出版物”成立。
- 本卡必须原创，不引用原作句子；输出笔记页+冷刀句+留白规则。

# 案例能力牌：CALM KNIFE / 冷刀笔记（Joan Didion 向）

## 1) 归属牌组（固定掉落载体）
**牌组：文档系｜笔记出版牌组（Notebook + Knife Sentence + Ellipsis）**  
固定掉落（不可漂移）：
- 笔记页主视觉
- 三段笔记
- Knife Sentence + 留白规则
- 桌面/出版页存在场景

---

## 2) director_spec（0–2 个按钮 + 默认路径）
- 按钮1：距离：旁观（默认）/ 在场
- 按钮2：刀口：隐喻（默认）/ 直切
- 默认：旁观 + 隐喻

---

## 3) visual_spec（主视觉 Image + 存在场景 Context Shot）

### 3.1 主视觉（Image）提示词
```text
照片小贴附（裁切避开人脸/门牌）；三段文本（Observation/Dialogue fragment/Index to self）；底部Knife Sentence≤20字；声明原创模拟不引用原作。
```

### 3.2 存在场景（Context Shot）提示词
```text
桌面摊开笔记本与折角稿纸，旁有铅笔与便签；可见本次笔记页缩略。
```

---

## 4) artifacts（主展品候选 2–4：名称 + 详细提示词）
### Artifact A｜Notebook Triptych
```text
三段各40–70字：观察/对话碎片/自我索引；禁止引用原句。
```
### Artifact B｜Knife Sentence
```text
≤20字断言式，冷静不鸡汤。
```
### Artifact C｜Ellipsis Rule
```text
2条：永远不写什么；为什么不写（让读者补完）。
```

---

## 5) 一次完整 Session（Scan → Load → Direct → Reveal → Collect）
Scan：写进笔记 → Load：距离/刀口 → Direct：2按钮 → Reveal：先冷刀句后笔记 → Collect：主展品=冷刀句。

---

## 6) 安全与边界（必须遵守）
- 输出必须原创，不引用原句。
- 不写真实姓名/地址。
- 保持克制，避免猎奇。
