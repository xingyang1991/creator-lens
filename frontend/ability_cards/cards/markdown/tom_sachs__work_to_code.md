# Tom Sachs｜WORK TO CODE / 规程手册包（案例级）

> 关键气质：**工坊、流程、QA、手工与制度**。  
> 核心方法：把图像转为“可复刻的制作规程”。

## 0) 三件套
1. **原型物作品照（Hero Object）**
2. **Knolling 工具排布现场（Workbench Layout）**
3. **制作单+质检清单（Build Sheet + QA）**

## 1) 归属牌组
- **卷宗系｜规程手册包牌组（Workshop Manual）**

## 2) Director_spec
- 风格：`军规 / 航天 / 极客手册`
- 颗粒度：`8步 / 16步`
- 证书：`要 / 不要`
- 风险：`不包含危险制造；不包含武器/爆炸物；不抄袭品牌设计`

## 3) Visual_spec
- 工坊桌面、胶带、标签、编号、签字栏
- 质检清单像飞行前检查

## 4) Artifacts
1. **Build Sheet**（步骤+时间+工具）
2. **QA Checklist**（勾选+签名）
3. **材料清单**（字段化）
4. **展出条件**（光线/距离/警示）

## 5) Workflow
1. 识别：可被“拆解成步骤”的对象/场景  
2. 配牌：规程手册包为主；辅以“研发工坊页/索引卡/临床陈列柜”  
3. 导演：默认安全；任何危险制作内容直接禁用  
4. 生成：先做制作单，再做视觉排布  
5. 交付：输出可用模板
